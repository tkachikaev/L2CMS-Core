# Модули

Модули являются доверенным PHP-кодом, а не изолированными плагинами. Только владелец может устанавливать, включать, подтверждать новую версию, применять миграции, отключать и удалять модуль. Администратор видит состояние модулей, редактор не имеет доступа к их управлению.

Каждый модуль использует строго проверяемый manifest, неизменяемую историю миграций, scoped routes, переводы, views и необязательные пункты навигации. Изменение или удаление уже применённой миграции блокирует runtime-загрузку до решения владельца. Неудачный пакет миграций откатывает только собственные новые изменения.

Встроенный модуль `promo-codes` добавляет одну или несколько привязанных к GameServer наград в основной веб-инвентарь. Отключение или удаление кода сохраняет историю активаций и наград.

Установка ZIP через браузер, автоматическое удалённое обновление модулей и sandbox пока намеренно отсутствуют.
