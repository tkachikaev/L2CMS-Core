# Документация KaevCMS

KaevCMS — CMS на Laravel для серверов Lineage II. Здесь описаны публичный сайт, личный кабинет игрока, административная панель, подключения игровых баз, статистика, веб-инвентарь, модули, шаблоны, почта и ручное веб-обновление.
