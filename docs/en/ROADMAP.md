# Roadmap

Current priority is release hardening: installation, upgrades, documentation, diagnostics, and hosting compatibility.

Planned after public feedback:

- donation-provider modules such as Pally;
- additional game-server schema profiles and community-tested drivers;
- safer module distribution and update workflows;
- expanded hosting diagnostics;
- optional convenience tooling without weakening the private-core deployment model.

The reward queue remains intentionally neutral: KaevCMS writes safe queue rows, while the server owner chooses the delivery consumer.
