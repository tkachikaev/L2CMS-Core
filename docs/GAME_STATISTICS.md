# Public statistics / Публичная статистика

## English

Statistics are enabled per GameServer with independent ranking limits, caching, and failure cooldowns.

Read the current guide: [GAME_INTEGRATION.md](en/GAME_INTEGRATION.md).

## Русский

Статистика включается отдельно для GameServer, использует независимые лимиты рейтингов, кеш и cooldown при сбое.

Актуальная инструкция: [GAME_INTEGRATION.md](ru/GAME_INTEGRATION.md).
