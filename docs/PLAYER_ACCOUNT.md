# Player account / Личный кабинет игрока

## English

The account uses a persistent Livewire shell with separate characters, game accounts, profile, and web inventory pages.

Read the current guide: [ADMINISTRATION.md](en/ADMINISTRATION.md).

## Русский

Личный кабинет использует постоянную Livewire-оболочку и отдельные страницы персонажей, игровых аккаунтов, профиля и веб-инвентаря.

Актуальная инструкция: [ADMINISTRATION.md](ru/ADMINISTRATION.md).
