# Game assets / Игровые изображения

## English

Owner-provided item and character images live under `public/uploads/game-assets`; server-specific files override common files.

Read the current guide: [GAME_INTEGRATION.md](en/GAME_INTEGRATION.md).

## Русский

Иконки предметов и аватары персонажей владельца находятся в `public/uploads/game-assets`; серверные файлы имеют приоритет над общими.

Актуальная инструкция: [GAME_INTEGRATION.md](ru/GAME_INTEGRATION.md).
