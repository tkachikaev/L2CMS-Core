# CMS users / Пользователи CMS

## English

Administrators can search, inspect, disable, re-enable, and send account notifications without deleting users.

Read the current guide: [ADMINISTRATION.md](en/ADMINISTRATION.md).

## Русский

Администраторы могут искать, просматривать, блокировать, включать и отправлять уведомления без удаления пользователя.

Актуальная инструкция: [ADMINISTRATION.md](ru/ADMINISTRATION.md).
